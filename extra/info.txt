This direcotry contains extra stuff, mainly for Win32 builds.

flex
	CPP code for the object loader after flex has been run, and modified to run on msvc++2010

include
	Headers for SDL, confuse, zziplib, etc. for msvc++2010

lib
	Libraries and DLLS for SDL, confuse, zziplib, etc. for msvc++2010

dlls
	Some old cross-compiled DLLs
