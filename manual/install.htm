Install

  • Download the zip.
  • Extract it somewhere, doesn't matter where.
  • Run "Chaotic Rage.exe".

System Requirements

  • OS
    • XP SP3, Vista or 7
    • Ubuntu 10.04
    • Wine, use winetricks for vc2005 and vc2010
  • Video
    • OpenGL 2.0 or later

If you have issues

Errors about MSVCP100.dll or MSVCR100.dll
Install the C++ Redist 2010 from MS website

Errors about side-by-side configuration
Install the C++ Redist 2010 from MS website
and the C++ Redist 2005 SP1 from MS website. Grab the x86 one.

Errors about application configuration
Install the C++ Redist 2010 from MS website
and the C++ Redist 2005 SP1 from MS website. Grab the x86 one.

Errors about DecodePointer
You need to update to a more recent service pack of Windows XP (service pack 3)
